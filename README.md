# 文章管理系统（AMS）
基于PHP+AJAX开发的轻量级文章管理系统，支持文章发布（含附件/配图上传）、编辑、删除、收藏、评论及用户管理等核心功能。

## 技术栈
- 后端：PHP
- 前端：HTML/CSS/JS + jQuery
- 前后端交互：AJAX
- 数据库：MySQL

## 核心功能
1. 文章管理：发布（支持附件/配图上传）、编辑、删除、收藏，支持文章搜索
2. 用户管理：登录/退出、密码修改、管理员权限控制
3. 评论功能：文章评论提交与展示
4. 后台管理：管理员仪表盘、用户/文章数据统计

## 运行步骤
1. 环境要求：PHP 7.0+、MySQL 5.7+、Apache/Nginx
2. 新建MySQL数据库，导入sql/ams.sql（包含表结构+示例数据）
3. 修改php/database.php中的数据库连接信息（替换占位符为你的账号/密码）
4. 将项目放到服务器根目录（如XAMPP的htdocs），访问index.html即可运行
